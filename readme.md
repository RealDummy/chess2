# How To Use
## Arguments
The program accepts up to one argument, a fen string for the starting position. If not provided, the starting position is used.
## IO Commands
* fen [fen string | show*]
* perft [n]
* show
* [UCI move]
* eval*

*not finished yet!

## About
Chess move validator.
